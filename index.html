안드로이드 전용 APK 어플 파일

아래 다운로드 버튼을 눌러 APK 설치 파일을 스마트폰에 직접 받아 실행할 수 있습니다.

📌 안드로이드 앱 설치 순서 (1분 완료)

  1. 하단 [안드로이드 APK 다운로드] 버튼 클릭
  2. 스마트폰에서 다운로드된 창고의민족.apk 실행
  3. '출처를 알 수 없는 앱 설치' 경고가 뜨면 [설정 ➔ 이 출처 허용] 켜기
  4. 설치 완료 후 바탕화면에서 창고의민족 앱 터치!
안드로이드 APK 다운로드 바로가기 ↗

💡 구글 드라이브 연결 시 [다운로드] 및 [그래도 다운로드]를 눌러주시면 바로 설치됩니다.

아이폰 3초 풀스크린 앱 등록

Safari 브라우저에서 [홈 화면에 추가]만 누르면 브라우저 틀이 사라진 진짜 아이폰 어플로 등록됩니다!

📌 아이폰 Safari 3초 설치 방법

  1. 아이폰 기본 브라우저 Safari(사파리)로 본 사이트에 접속
  2. 하단 중앙의 [공유 ⬆️] 아이콘 버튼을 터치
  3. 메뉴를 아래로 내려 [홈 화면에 추가] 클릭
  4. 우측 상단 [추가] 클릭 후 홈 화면의 '창고의민족' 앱 아이콘 실행!
